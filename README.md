# teameasy
Suivi et automatisation de procédés
Premiére etape : installer python sur ta machine
Deuxiéme étape : installer une variable d'environnement python
Troisiéme étape : activer la variables d'environnement
Quatriéme étape : installer le contenu du requirement
Cinquiéme étape : installer postgresql
6ieme étape : créer une base de données postgreSQL
7eme étape : paramétrer la base postgresql dans le programme depuis le fichier .env
8ieme étape : lancer la commande python manage.py makemigrations definir les propriétés de la base
9ieme étape : lancer la commande python manage.py migrate initialiser la base
10ieme étape : lancer la commande python manage.py runserver lancer le server test
